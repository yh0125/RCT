# 统计分析计划（Statistical Analysis Plan）

## AI辅助放射报告可视化解读RCT

**版本：** 2.0  
**日期：** 2026年3月28日  
**参考方法学：** Prucker et al. (Radiology, 2025)

---

## 1. 分析原则

### 1.1 分析人群

| 人群 | 定义 | 用途 |
|------|------|------|
| ITT（意向性治疗） | 所有经随机化分组的受试者 | 主要分析 |
| Per-Protocol | 完成全部评估且无重大方案偏离 | 敏感性分析 |

### 1.2 显著性水平

- 单个条目分析：双侧 P < 0.05
- 三个域复合评分的多重比较：Bonferroni校正，α = 0.05/3 = **0.0167**
- 所有检验均为双侧

### 1.3 缺失数据

- 若无缺失，执行全病例分析（参考 Prucker et al. 200例全部完成）
- 若存在缺失：
  - 缺失率 < 5%：完整病例分析
  - 缺失率 ≥ 5%：多重插补（MI）作为敏感性分析

## 2. 基线特征

### 2.1 描述性统计

| 变量类型 | 报告方式 |
|---------|---------|
| 正态连续变量 | 均值 ± 标准差 |
| 非正态连续变量 | 中位数（IQR） |
| 分类变量 | 频数（百分比） |

### 2.2 组间均衡性检验

| 变量类型 | 检验方法 |
|---------|---------|
| 连续变量 | Mann-Whitney U 检验 |
| 分类变量 | χ² 检验（期望频数 < 5时用Fisher精确检验） |

报告的基线变量：年龄、性别、最高学历、自评健康状况、检查类型、检查部位、是否首次检查、基线担忧水平。

## 3. 主要结局分析

### 3.1 Likert量表条目

#### 分析方法：比例优势Logistic回归（Proportional Odds Logistic Regression）

对9个7点Likert条目（CW1-CW3, TC1-TC3, RP1-RP3）分别建模：

**模型公式：**
```
logit[P(Y ≤ j)] = αj - (β₁·Group + β₂·Age + β₃·Gender + β₄·Education 
                        + β₅·HealthStatus + β₆·BaselineConcerns)
```

其中 j = 1, 2, ..., 6（7个类别产生6个截断点）

**报告内容：**
- 粗比值比（Crude OR）及95%CI
- 调整后比值比（Adjusted OR）及95%CI
- P 值

**调整变量：**

| 变量 | 类型 | 参考组 |
|------|------|-------|
| 分组（Group） | 二分类 | A组（文字解读） |
| 年龄（Age） | 连续或二分类（以中位数为界） | 高年龄组 |
| 性别（Gender） | 二分类 | 男性 |
| 最高学历（Education） | 多分类 | 大学及以上 |
| 自评健康状况（Health Status） | 多分类 | 极好 |
| 基线担忧（Baseline Concerns） | 多分类 | 极多担忧 |

**模型假设验证：**
- Brant 检验验证比例优势假设
- 若不满足，考虑广义有序logit模型或非参数方法

#### 认知负荷域的解释

认知负荷条目（CW1-CW3）的评分方向与其他域相反：
- 低评分 = 低认知负荷 = 更好
- 若图文组优于文字组，预期 OR < 1

#### 文本理解域和报告感知域的解释

- 高评分 = 更好的理解/感知
- 若图文组优于文字组，预期 OR > 1

### 3.2 复合评分（Composite Scores）

#### 步骤

1. **内部一致性：** 计算各域（CW, TC, RP）三条目的 Cronbach's α
2. **阈值检查：** 仅当 α ≥ 0.70 时生成复合评分
3. **生成复合评分：** 各条目先z标准化，然后取均值
4. **分析：** 对复合评分使用与单条目相同的比例优势logistic回归
5. **多重比较校正：** 三个域的复合评分比较应用Bonferroni校正（α = 0.0167）

### 3.3 阅读时间

#### 分析方法：普通最小二乘回归（OLS Regression）

**模型：**
```
Reading_Time = β₀ + β₁·Group + β₂·Age + β₃·Gender + β₄·Education 
             + β₅·HealthStatus + β₆·BaselineConcerns + ε
```

**报告内容：**
- 粗β系数及95%CI
- 调整后β系数及95%CI
- P 值

## 4. 次要结局分析

### 4.1 担忧变化

- 使用比例优势logistic回归分析阅读后担忧评分（Q10）的组间差异
- 将基线担忧（Q0）作为协变量纳入
- 计算 ΔConcerns = Q10 - Q0，使用Wilcoxon秩和检验比较组间差异

### 4.2 视觉辅助评估（仅B组描述性分析）

- 对VA1-VA3三个条目进行描述性统计
- 报告各选项的频率分布
- 特别关注VA3（图片是否增加焦虑）的分布

### 4.3 满意度

- 对S1-S2条目使用比例优势logistic回归进行组间比较

### 4.4 AI内容质量（描述性）

- 报告事实错误、遗漏、不当添加的发生率及95%CI
- 按严重程度分级报告
- 报告评审者间ICC和加权Cohen's κ
- 报告临床有用性和总体质量评分的分布

### 4.5 可读性指标（仅对AI文字解读）

- 对比原始报告与AI解读的可读性指标差异（配对t检验）：
  - Flesch-Kincaid Grade Level
  - Flesch Reading Ease
  - SMOG (Simple Measure of Gobbledygook)
  - Gunning Fog Index
  - Coleman-Liau Index
- 对比AI解读的字数减少（Wilcoxon符号秩检验）

## 5. 亚组分析

针对每个协变量进行分层分析，使用与主要分析相同的回归模型，
报告各亚组的粗OR和调整后OR：

| 分层变量 | 分组方式 |
|---------|---------|
| 年龄 | < 中位数 vs ≥ 中位数 |
| 性别 | 男 vs 女 |
| 学历 | 6个类别（参考组：大学及以上） |
| 自评健康 | 7个类别（参考组：极好） |
| 基线担忧 | 7个类别（参考组：极多担忧） |
| 检查类型 | CT vs MRI vs X光 |

## 6. 敏感性分析

| 分析 | 方法 | 目的 |
|------|------|------|
| Per-Protocol分析 | 排除方案偏离者 | 评估方案依从性的影响 |
| 无调整分析 | 仅纳入分组变量 | 评估协变量调整的影响 |
| 非参数分析 | Mann-Whitney U检验 | 不依赖回归模型假设 |
| 中期分析 | 50%入组后的条件效能 | 评估效应量假设 |

## 9. 报告标准

本研究的统计结果将遵循以下报告标准：
- **CONSORT 2010** 声明（随机对照试验）
- **SPIRIT 2013** 声明（试验方案）
- 所有 P 值报告至小数点后三位或 < .001
- 所有效应量报告 95%CI
