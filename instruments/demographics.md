## 数据编码

| 变量名 | 描述 | 编码 |
|--------|------|------|
| id | 受试者编号 | 字符串 |
| age | 年龄 | 连续（岁） |
| sex | 性别 | 1=女, 2=男, 3=其他 |
| education | 最高学历 | 1=未受正式教育, 2=初中及以下, 3=高中/中专/技校, 4=职业培训, 5=大专, 6=大学本科及以上 |
| health_status | 自评健康 | 1-7 |
| exam_type | 检查类型 | 1=CT, 2=MRI, 3=X光, 4=其他 |
| exam_site | 检查部位 | 分类编码 |
| first_exam | 是否首次 | 0=否, 1=是 |
| exam_reason | 检查原因 | 分类编码 |
| informed_by_doc | 是否已被告知 | 1=详细告知, 2=简要提及, 3=未沟通 |
| group | 分组 | 0=A组(文字), 1=B组(图文) |
